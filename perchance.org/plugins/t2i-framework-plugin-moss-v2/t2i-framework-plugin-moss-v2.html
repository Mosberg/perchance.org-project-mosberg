New in v2

Built-in mechanics and configurable styling for generated pages.

  • Theme presets: aurora, sunrise, slate
  • Quick actions row (generate, clear output, clear prompt, scroll top)
  • Optional sticky generate bar
  • Keyboard shortcuts: Ctrl/Cmd+Enter to generate, Ctrl/Cmd+K focus prompt
  • Session stats, scroll-top button, toast feedback, and image hover polish
